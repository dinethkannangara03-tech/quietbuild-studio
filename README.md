# QuietBuild Studio Pro

A premium, responsive React + Vite portfolio for QuietBuild Studio.

## Edit website content

Edit `src/data/siteData.js` to change the studio details, links, hero content,
projects, services, process steps, page copy, and footer text.

Add real image files to `public/images/`. Every image path is set in
`src/data/siteData.js`, so filenames can be changed there.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Routes

- `/`
- `/work`
- `/services`
- `/about`
- `/contact`

## Assets and links to replace

Real visual assets belong in `public/images/`. See `public/images/README.md` for the exact default filenames.

Replace the placeholder email, WhatsApp number, GitHub URL, social URLs, live demo links, and case-study placeholders before launch. Connect the contact form to a backend before enabling its submit button.
